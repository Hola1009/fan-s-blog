### 表格

1. 表格表示为单元格竖线用`|`表示，表头与表格内容之间需要`---|---`类似的格式隔开(只有添加了这样的分割线才会被识别为表格排版格式)。
2. 表格对齐格式如下
3. 左对齐（默认）：`:----`
4. 右对齐：`----:`
5. 居中对齐：`:----:`
6. 块级元素（代码区块、引用区块）不能插入表格中,但是链接可以插入。


| a      |   b    |       c |     |
| :----- | :----: | ------: | --- |
| dddddd | eeeeee | fffffff |     |
| g      |   h    |       i |     |
| s      |   h    |       c |     |

| a | b |  |  |
| :--- | :--- | :--- | ---- |
|  |  |  |  |
|  |  |  |  |
