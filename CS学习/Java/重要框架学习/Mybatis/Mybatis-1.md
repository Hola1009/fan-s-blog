## 什么是 MyBatis 
MyBatis 是 一款优秀[[#持久层]]框架, 用于简化 JDBC 开发 
## JDBC 的 缺点
1. 硬编码
	1. 注册驱动获取链接
	2. SQL语句
![[Pasted image 20240417191110.png|left|400]]
2. 操作繁琐
	1. 手动设置参数
	2. 手动封装结果集
![[Pasted image 20240417191144.png|left|400]]

mybatis 几乎免除了所有 JDBC 的操作,  其直接将 硬编码写道配置文件中去, 要用的时候读取即可
还免除了遍历获取结果集的工作
