# 什么是 Selenium 
Selenium是一个做UI自动化测试的工具
# Selenium 工作原理
角色: 代码, 驱动, 浏览器
驱动: 有了驱动就可以让代码操作浏览器

Selenium 是一个 浏览器驱动, 
自动化程序 创建http请求 会有 驱动的 http server 去接收请求
然后驱动再去操控浏览器, 执行操作步骤, 
浏览器将执行结果返回给驱动, 驱动将结果再返回给 自动化脚本
